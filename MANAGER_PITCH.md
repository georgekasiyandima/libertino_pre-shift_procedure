# Manager Pitch Guide - Libertino Pre-Shift Briefing System

## Executive Summary

A comprehensive digital briefing system designed to **reduce bottlenecks during rush times**, **improve table turnover by 10-30%**, and **enhance team communication** through real-time updates and coordination tools.

## Problem Statement

**Current Challenges:**
- Bottlenecks during peak hours reduce table turnover
- Lack of real-time communication between staff
- Difficulty tracking table status and reservations
- No centralized system for daily updates (salad, dessert, specials)
- Staff waste time searching for items (napkins, candles, etc.)
- No visual representation of seating arrangements

## Solution Overview

A **mobile-friendly web application** that provides:
1. **Real-Time Floor Communication** - Instant updates during rush
2. **Admin Panel** - Easy updates for daily specials and floor information
3. **Visual Floor Plan** - Real-time table status and seating arrangements
4. **SOS Messaging** - Team communication for urgent needs
5. **Item Location Finder** - Quick access to where things are stored
6. **Table Turn Optimization** - Data-driven insights to improve service

## Key Features Demonstration

### 1. Admin Panel (`/admin`)
**What it does:** Centralized control for all briefing updates

**Show:**
- Update Salad of the Day (30 seconds)
- Update Dessert of the Day (30 seconds)
- Post real-time floor messages
- Add item locations for team reference
- Create SOS messages for urgent needs

**Benefit:** No more whiteboards or printed sheets - everything digital and instant

### 2. Real-Time Floor Communication
**What it does:** Live updates visible to all staff

**Show:**
- Post a floor message: "Table 12 needs attention"
- Show it appears instantly on all devices
- Demonstrate SOS messaging system
- Show item location finder

**Benefit:** Everyone stays informed, reduces wait times, improves coordination

### 3. Visual Seating Arrangement
**What it does:** Clear diagram of all bookings by time and section

**Show:**
- Filter by time slot (6:00 PM, 7:30 PM, etc.)
- Filter by section (MDA, Outside, Library)
- See guest names, party sizes, special requests
- Visual status (Available, Reserved, Occupied)

**Benefit:** Better table management, reduced confusion, smoother service

### 4. Table Turn Optimization
**What it does:** Data-driven insights to improve efficiency

**Show:**
- Peak hours awareness (when to expect rush)
- Section capacity tracking (which areas need attention)
- Service timing guidelines (target times for each stage)
- Current bottlenecks identification

**Benefit:** Proactive management, reduced bottlenecks, improved turnover

**Assumptions:**
- Average table value: R500
- Current covers per night: 80
- Target improvement: 20% (16 additional covers)

**Monthly Revenue Increase:**
- 16 additional covers × R500 = R8,000 per night
- R8,000 × 30 nights = **R240,000/month**
